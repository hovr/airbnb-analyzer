# Airbnb Analyzer

Airbnb Analyzer is a Chrome extension that helps you review Airbnb listings before you book.

It collects listing details, amenities, and reviews from an Airbnb wishlist or a single Airbnb listing, then gives you one prompt you can copy or send to ChatGPT or Claude.

## ✨ What it does

- Works on Airbnb wishlist pages
- Works on individual Airbnb room/listing pages
- Collects useful listing details and guest reviews
- Builds a prompt you can copy and paste manually, or send directly to ChatGPT or Claude
- Remembers whether you last chose ChatGPT or Claude
- Spots possible issues, patterns, or red flags before booking
- The AI assistant will be aware of property details and all reviews so you can quiz it about any requirements/issues.
- It will also try to give you the best and worst of wishlist listings (you can ask it to re-assess the lists based on your own criteria).

## 🚀 How to install it

1. [Download the ZIP file for this extension](https://github.com/hovr/airbnb-analyzer/archive/refs/heads/main.zip).
2. Open the ZIP file and move the extracted folder somewhere it can stay (it needs to stay here for the extension to work).
3. Open Chrome and go to **Settings > Extensions**, or copy this into the address bar and press Enter:

   `chrome://extensions`

4. Turn on **Developer mode** in the top-right corner.
5. Click **Load unpacked**.
6. Select the extracted folder (it may be named `airbnb-analyzer-main`).
7. The extension should now appear in Chrome.
8. Click the puzzle-piece icon in Chrome and pin the extension.

## 🏡 How to use it

### ❤️ Analyze a wishlist

1. Open your Airbnb wishlist page.
2. Make sure the listings you want to check are visible.
3. Click the Airbnb Analyzer extension icon.
4. Click **Start Analysis**.
5. Leave the browser & tabs alone and in view while it works (go make a nice brew ☕️).
6. When it finishes, click **Copy Prompt** if you want to paste the prompt manually into any AI assistant.
7. Or use **Send to**, choose **ChatGPT** or **Claude**, then click **Send**.
8. The extension opens the selected assistant and tries to place the prompt in the chat box. It also copies the prompt to your clipboard as a fallback.

### 🔎 Analyze one listing

1. Open an Airbnb listing page.
2. Click the Airbnb Analyzer extension icon.
3. Click **Start Analysis**.
4. When it finishes, click **Copy Prompt** if you want to paste the prompt manually into any AI assistant.
5. Or use **Send to**, choose **ChatGPT** or **Claude**, then click **Send**.
6. The extension opens the selected assistant and tries to place the prompt in the chat box. It also copies the prompt to your clipboard as a fallback.

## ⏳ What to expect

The extension may open property tabs while it gathers details and reviews. This can take a few minutes, especially for long wishlists or listings with lots of reviews.

Try not to close or change the tabs while the extension is working and leave the browser in view!

When sending a prompt to ChatGPT or Claude, the selected site must be available in your browser. If the prompt does not appear automatically, paste it manually; the extension copies it before opening the assistant.

## 🔄 Updating the extension

When a newer version is available, the popup will show an update notice.

To update:

1. Download the latest ZIP file.
2. Replace your old extension folder with the new extracted folder.
3. Open Chrome and go to **Settings > Extensions**.
4. Click the reload button on the Airbnb Analyzer card.
